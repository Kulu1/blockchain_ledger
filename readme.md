This is a simple blockchain ledger simulation that shows simple blockchain concepts


What this project demonstrates

This version covers the core concepts:
    Block structure
    Hash generation with SHA-256
    Hash linking
    Proof of Work
    Pending transactions
    Mining reward
    Blockchain validation
    Basic balance calculation


This is a learning project, not a production blockchain, because it does not include:
    digital signatures
    wallets with public/private keys
    distributed peer-to-peer networking
    consensus across multiple nodes
    persistence in a database
